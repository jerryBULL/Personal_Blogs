CSS權重(CSS Specificity)

前言

權重的意思就是CSS的優先權。

相同權重的情況下後寫的CSS可以覆蓋先寫的CSS。

當兩個選擇器同時作用在一個元素，權重高的優先生效。

基本權重大小

inline style > ID > Class = psuedo-class(偽類) = attribute（屬性選擇器）> Element > *

各個權重：

*

權重為0-0-0-0。

全站預設值。

Element

Element種類很多，例如：div,ul,li,ol....等，可查看HTML元素。

attribute（屬性選擇器）

權重為0-0-1-0。

例如：[type=email],[type:checkbox],[attr]...等等

psuedo-class(偽類)

權重為0-0-1-0。

:link,:hover,:focus等。

class

權重為0-0-1-0。

ID

權重為0-1-0-0。

inline style

權重為1-0-0-0。

直接寫在Html行內的style，例如：

<div style="background:black">

</div>

最後還有一個最強王者"!important"且只有"!important"自己可以超越。

只要在任何一段css的語法後面加入!important，想要覆蓋的話就必須在後來寫的css後面也加!important才可覆蓋過去。
